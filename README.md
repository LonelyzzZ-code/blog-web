# Blog Web

基于 **Vue 3 + TypeScript + Vite** 的博客前端，对接 blog-api 后端。

## 技术栈

- Vue 3（Composition API）
- TypeScript
- Vue Router
- Axios
- Vite

## 功能

- 文章列表（分页）
- 文章详情（含访问量自增）
- 评论列表 + 发表评论 + 点赞
- 访问量排行榜

## 启动

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 确保后端已启动（http://localhost:8000）
```

然后浏览器打开 http://localhost:5173
